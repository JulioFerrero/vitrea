# @vitrea/create

Scaffold a new Vitrea project.

## Usage

```bash
pnpm dlx @vitrea/create
```

Create into the current folder:

```bash
pnpm dlx @vitrea/create .
```

Preview without writing files:

```bash
pnpm dlx @vitrea/create --preview
```

## After Scaffolding

```bash
pnpm install
pnpm setup
pnpm dev
```
